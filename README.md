### `npm version + node version`
nvm 은 따로 안쓰겠음.
node version - v14.15.4
npm version - v6.4.10

### `npm install`
package.json + package-lock.json 에 만들어진 종속성 트리 대로 node_modules 구성해주는것.

### `npm start`
FE 실행시키기. 아마도 포트 3000.

### `npm test`
이건 나도 안해봄.

### `npm run build`
나중에 배포할때 SPA 로 만들어주는것.

### `npm run eject`
모르겠음.
